# JavaScript - Arrays and Functions

## Description
This mini-project demonstrates how to pass arrays as arguments into various types of JavaScript functions to perform mathematical operations.

Per the assignment instructions, the script utilizes three different styles of function syntax:
1. **Anonymous Function Expression (`findMax`):** Iterates through the array to find and return the maximum number.
2. **Arrow Function (`calculateSum`):** Utilizes a `for...of` loop to calculate the sum of all elements in the array.
3. **Standard Function Declaration (`countOdds`):** Evaluates each element using the modulo operator to count how many odd numbers exist in the array.

## How to Run
1. Unzip the project folder.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same directory.
3. Open `index.html` in your web browser.
4. Press `F12` (or right-click and select "Inspect") to open the browser's Developer Tools.
5. Click on the **Console** tab to view the results applied to the test array `[4, 8, 2, 11, 6, 7, 10]`.
