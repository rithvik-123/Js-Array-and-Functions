# My Learning Journey - Array Task

## Description
I built this project to practice writing functions that process array data.

## Challenges
* I initially struggled with the `countOdds` function because I was confused about how to reset my counter.
* I realized that using `let` for my variables was necessary so I could update them inside the loops.
* I chose these specific loop styles because I find them easier to read than the modern arrow function shorthand.

## How to Run
1. Open `index.html` in your browser.
2. Press F12 to open the console and view my function outputs.
